# Machine and Deep Learning for S&P 500

#Objective
Investigate the difference in performance of deep learning models of varying complexity for predicting the s&p 500. In particular, Gated Recurrent Units vs Recurrent Neural Networks

#Tools 
Python, Numpy, YahooFinance, Excel, Pandas

#Dataset
S&P 500 closing prices from 1990-2019

#Findings
Gated Recurrent Units much better at classifying into the Buy, Hold, Share classes than its competitor. Recurrent Neural Network collapsed under the Sell class. Evidence of underfitting which was surprising, suggests economic variables not indicative enough (could improve this), maybe not enough data points due to limited computational power
